# Email Spam Classifier

A Machine Learning project that classifies emails as Spam or Ham using the Naive Bayes algorithm.

## Features

* Email classification (Spam/Ham)
* Data preprocessing
* Text vectorization using CountVectorizer
* Naive Bayes classification
* Accuracy evaluation

## Dataset

* Kaggle Spam Email Dataset

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Algorithm

* Multinomial Naive Bayes
